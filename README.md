Student-Resource Recommender

A Knowledge-Based Recommendation System for Modern Student Learning.

 Project Overview :
The Stu-Resource Intelligent Agent is a Python-based tool designed to solve the problem of "information overload". 
In an era where educational content is fragmented across countless platforms, this agent acts as a centralized digital mentor. 
It filters high-quality resources in specialized fields—ranging from AI in Finance to Personality Development—based on the user's specific learning preferences.


 Key Features:
1. Interdisciplinary Scope: Provides curated paths for Business with AI, Finance with AI, Robotics, Soft Skills, and Mindset growth.

2. Dual-Track Learning: Users can choose between Visual Learning (YouTube) or Text-Based/Interactive Learning (Websites).

3.Intelligent Agent Logic: Uses a knowledge-based approach to map user "percepts" (inputs) to the most rational educational "action" (recommendation).

4. Dynamic Curation: Implements randomization to ensure a diverse range of resource suggestions, preventing a static user experience.

5. Minimalist UI: Features a professional, distraction-free terminal interface with a modern aesthetic.
   

Technical Implementation:

The project is built entirely in Python and focuses on clean, modular code.

Language: Python 3.x

Logic: Knowledge-Based Filtering

random: Ensures diversity in recommendations.

time: Manages the pacing of the interaction for a better user experience.


 Installation & Usage :

1. Running the Agent
Clone this repository or download the resource_agent.py file, then run:
python resource_agent.py

2. How to Use-
   
Launch: Start the script to meet your AI Learning Assistant.

Select Field: Choose from the list of specialized subjects (e.g., Robotics, Finance with AI).

Choose Medium: Indicate if you prefer YouTube channels or online websites.

Explore: Receive your personalized top pick and alternative resources instantly.


 Design Philosophy:
 
This project follows a Purposeful & Minimalist design. Rather than building a complex system with low-quality data, this agent prioritizes Curated Excellence. Every resource in the knowledge base has been hand-selected to ensure it provides genuine value to a student's career and personal growth.

 
 Future Roadmap:
 
API Integration: Connecting to the YouTube Data API for real-time trending content.

User Profiles: Implementing a local database to save a student's learning history and track progress.

Machine Learning: Transitioning from a Knowledge-Based system to a Collaborative Filtering model that learns from a community of student users.
